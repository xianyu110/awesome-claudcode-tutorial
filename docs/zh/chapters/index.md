# 课程章节

欢迎来到 Claude Code 课程式学习路径！

本教程采用**系统化章节**组织，共 29 个章节，从基础入门到企业级应用，循序渐进地掌握 Claude Code。

## 📚 学习方式

### 🎓 课程式学习（当前）
- **特点**：结构清晰、循序渐进、实战导向
- **适合**：初学者和希望系统学习的进阶者
- **内容**：29 个精心设计的章节

### 📖 文章式学习
- **特点**：全文搜索、主题分类、细致入微
- **适合**：需要快速查阅特定主题的开发者
- **内容**：212+ 篇详细文章
- **链接**：[前往文章教程](/zh/articles/)

## 🎯 课程结构

### 第一部分：环境搭建与基础交互
本部分介绍 Claude Code 的基础知识和安装配置。

- [01. 安装 Claude Code](./01-installation.md) - 多平台安装指南
- [02. 登录与授权](./02-authentication.md) - 账户设置与认证
- [03. 第一个实战问题](./03-first-task.md) - 动手实践
- [04. 三种模式详解](./04-modes.md) - 默认/自动/规划模式

### 第二部分：复杂任务处理与终端控制
学习如何使用 Claude Code 处理复杂的开发任务。

- [05. 执行终端命令](./05-terminal-commands.md) - Bash 工具使用
- [06. 使用规划模式](./06-plan-mode.md) - Plan Mode 深入讲解
- [07. 跳过权限检测](./07-permissions.md) - 权限管理技巧
- [08. 后台任务管理](./08-background-tasks.md) - 并行任务处理

### 第三部分：多模态与上下文管理
探索 Claude Code 的高级功能。

- [09. 版本回滚 (Rewind)](./09-rewind.md) - 会话状态管理
- [10. 图片处理](./10-image-processing.md) - 多模态能力
- [11. 安装 MCP Server](./11-mcp-setup.md) - MCP 集成
- [12. 恢复历史会话](./12-resume.md) - 会话恢复
- [13. 使用 MCP 工具还原设计稿](./13-mcp-design.md) - 设计到代码
- [14. 上下文压缩与清除](./14-context-management.md) - 上下文优化
- [15. 项目记忆文件](./15-claude-md.md) - CLAUDE.md 项目记忆

### 第四部分：高级功能扩展与定制
学习如何扩展和定制 Claude Code。

- [16. Hook 系统](./16-hooks.md) - 自定义钩子
- [17. Agent Skills](./17-skills.md) - 可重用工作流
- [18. SubAgent](./18-subagents.md) - 并行任务处理
- [19. Skills vs SubAgents](./19-skills-vs-subagents.md) - 选择指南
- [20. Plugin 系统](./20-plugins.md) - 插件开发

### 第五部分：企业级应用与最佳实践 ⭐
**新增章节**：面向企业和团队的高级应用场景。

- [21. 工作流与最佳实践](./21-workflows-best-practices.md) - 代码审查、调试、重构工作流
- [22. 企业部署](./22-enterprise-deployment.md) - 托管设置、MDM策略、安全合规
- [23. 远程控制与会话](./23-remote-control-sessions.md) - SSH会话、云同步、多设备协作
- [24. 监控与可观测性](./24-monitoring-observability.md) - OpenTelemetry、Metrics、Logs、Traces
- [25. CI/CD 集成](./25-ci-cd-integration.md) - GitHub Actions、GitLab CI、Jenkins

### 第六部分：高级能力与多 Agent 协作 ⭐
**新增章节**：探索 Claude Code 的高级交互能力和多 Agent 协作模式。

- [26. Computer Use（屏幕操作）](./26-computer-use.md) - 屏幕识别、GUI 自动化、端到端测试

4. **社区交流**：在 GitHub Discussions 中分享学习心得
