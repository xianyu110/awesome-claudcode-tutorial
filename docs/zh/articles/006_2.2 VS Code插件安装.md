# 2.2 VS Code插件安装

## 2.2.1 安装VS Code

Claude Code 提供了 VS Code 插件，因此需要先安装 VS Code。

### 2.2.1.1 检查 VS Code 是否已安装

打开终端，运行以下命令：

## 检查 VS Code 版本
code --version

如果输出了版本信息，说明已安装，可以跳过后续步骤。

### 2.2.1.2 安装 VS Code

如果没有安装，可以通过以下方式安装：

#### 方式一：官方安装包（推荐）

1. 访问 VS Code 官方网站：https://code.visualstudio.com/
2. 下载适合你操作系统的安装包
3. 运行安装程序，按提示完成安装

#### 方式二：包管理器

**macOS（Homebrew）：**

brew install --cask visual-studio-code

**Linux（Ubuntu/Debian）：**

wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > packages.microsoft.gpg
sudo install -o root -g root -m 644 packages.microsoft.gpg /etc/apt/trusted.gpg.d/
sudo sh -c 'echo "deb [arch=amd64,arm64,armhf signed-by=/etc/apt/trusted.gpg.d/packages.microsoft.gpg] https://packages.microsoft.com/repos/code stable main" > /etc/apt/sources.list.d/vscode.list'
rm -f packages.microsoft.gpg
sudo apt update
sudo apt install code

**Linux（Fedora）：**

sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/yum.repos.d/vscode.repo'
sudo dnf install code

## 2.2.2 安装 Claude Code 插件

### 2.2.2.1 从 VS Code 市场安装（推荐）

1. 打开 VS Code
2. 点击左侧边栏的**扩展**图标（快捷键 `Ctrl+Shift+X` / `Cmd+Shift+X`）
3. 在搜索框中输入 `Claude Code`
4. 找到 Claude Code 插件，点击**安装**
5. 安装完成后插件自动启用

也可以直接点击 [VS Code 插件市场链接](https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code)，会自动跳转到 VS Code 安装。

### 2.2.2.2 手动安装插件

如果无法从市场安装，可以手动安装 VSIX 文件：

1. 前往 Claude Code 插件的 GitHub 仓库下载最新的 `.vsix` 文件
2. 打开 VS Code
3. 点击左侧边栏的**扩展**图标（快捷键 `Ctrl+Shift+X` / `Cmd+Shift+X`）
4. 点击扩展面板右上角的 **⋯**（更多操作）按钮
5. 选择**从 VSIX 安装...**
6. 选择下载的 `.vsix` 文件，点击安装
7. 安装完成后重新加载 VS Code

### 2.2.2.3 验证插件安装

1. 打开 VS Code
2. 点击左侧边栏的 **Claude Code** 图标
3. 或者按 `Ctrl+Shift+P` / `Cmd+Shift+P` 打开命令面板，输入 `Claude Code`
4. 如果看到相关命令，说明插件已成功安装

### 2.2.2.4 插件更新

插件会定期更新，提供新功能和修复。更新方式：

1. 打开 VS Code 扩展面板（`Ctrl+Shift+X` / `Cmd+Shift+X`）
2. 找到 Claude Code 插件
3. 如果有可用更新，会显示**更新**按钮，点击即可

或通过命令面板（`Ctrl+Shift+P` / `Cmd+Shift+P`）输入 `Extensions: Check for Extension Updates` 检查所有插件更新。
